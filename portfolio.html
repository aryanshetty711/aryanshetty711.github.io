Portfolio

I've listed some of the projects I've worked on. I've included what frameworks I've used, when I built them, as well as refernces where you can read more about them! I'm constantly striving to learn by working on new projects through participating in classes and events.

FastAid

A web app for patients with non serious ailments which can be used to find doctors in their area. App notified registered doctors within proximity with patient symptoms to set up remote/f2f consultation to reduce hospital congestion and minimize transmission risk.

  • Frameworks: Node.js, Javascript, Firebase, CSS, HTML
  • Where: HackTheWorld Hackathon
  • When: July'20
  • Link: Github , DevPost

Don't Panic!

Emergency notification app that serves as a panic button thereby informing emergency contacts of the victim's last known location. User is informed via call, text, and email. Primarily intended to serve those affected by natural disaster requiring immediate evacuation such as wildfire, tsunami etc.

  • Frameworks:Twilio, Google Maps API, SMTP, Python
  • Where: HackTX Hackathon
  • When: Oct'19
  • Link: DevPost

BOQ BUILDER

Python app that implemented a comprehensive and responsive GUI to finalize Citrix client purchases. Built to simplify the previously bloated process of manually generating the bill which required each employee to comb through system data.
Automated the generation and exporting of a Bill of Quantities to the GUI and Excel, thereby providing the client with a coherent summary of their order Established an easy to use interface to select desired product details as input Implemented search capability to drastically reduce processing time from 15 minutes to less than a minute, thereby saving the company over 7hrs per week

  • Frameworks: Python, Tkinter, XLSXwriter, Pandas
  • Where: Internship at Citrix Technologies
  • When: May'19 - July'19
  • Link: Github

Stock Predictor

  • Frameworks: Python, TensorFlow, BeautifulSoup
  • Where: TAMUhack'19
  • When: Jan'19
